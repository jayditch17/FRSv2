How to access the System:

1. Use Xampp to host the system. Download the application at "https://www.apachefriends.org/index.html" and install it at Drive C://. 
2. Once Xampp is installed, download the system at this link of the repository "https://github.com/jayditch17/FACILITY-RESERVATION-SYSTEM".
3. When the repository is downloaded, copy the folder FACILITY-RESERVATION-SYSTEM and paste it to C://xampp/htdocs.
4. Click on Xampp to open. 
5. Go to your browser and copy this link, "localhost/FACILITY-RESERVATION-SYSTEM/index.php" and login in with these credentials:
		email: superadmin@localhost.ph
		password: superadminfs
